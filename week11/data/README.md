# week11 data

Raw lab datasets for the week11 assignment and companion notebook. The notebook on Canvas references these files under the paths below.

## Layout

```
week11/data/
├── healthpulse/   (healthcare scenario — patient, lab, device data)
└── greenroute/    (logistics scenario — GPS, delivery, weather data)
```

Clone this repo locally to get the full copy, or stream a single file via:

```
https://raw.githubusercontent.com/prof-tcsmith/ism6562s26-class/main/week11/data/<scenario>/<path-to-file>
```
